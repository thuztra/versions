# versions

Machine-consumed update manifests for [Thuztra IDE](https://github.com/thuztra/thuztra-ide).

Do not edit by hand — this repo is written by the release workflow of
`thuztra/thuztra-ide` (`update_version.sh`, triggered on a `release: true`
dispatch of "Build - Windows"). Layout: `stable/win32/x64/{system,user,archive}/latest.json`.
